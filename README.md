# Форматер текста — PWA

## Структура файлов

```
formatter-pwa/
├── index.html       ← главная страница
├── manifest.json    ← PWA манифест (иконка, название, цвет)
├── sw.js            ← сервис-воркер (офлайн-режим)
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

## Как выложить (GitHub Pages — бесплатно)

1. Создайте аккаунт на github.com
2. Нажмите «New repository», назовите например `formatter`
3. Загрузите все файлы (включая папку icons/)
4. В настройках репозитория → Settings → Pages → Source: main branch
5. Через минуту приложение будет доступно по адресу:
   `https://ВАШ_НИК.github.io/formatter/`

## Как установить на Android

1. Откройте ссылку в Chrome на телефоне
2. Chrome покажет баннер «Добавить на главный экран»
   (или: меню ⋮ → «Установить приложение»)
3. Нажмите «Установить» — иконка появится на рабочем столе
4. Приложение работает офлайн после первого открытия

## Функционал

- Открытие .txt файлов с устройства
- Сохранение результата как .txt
- Вставка / копирование из буфера обмена
- Удаление двойных пробелов, лишних строк, обрезка краёв
- Верхний / нижний регистр
- Умное обнаружение и удаление специальных символов
- История изменений (отменить / вернуть)
- Работает офлайн
- Устанавливается как приложение
